# Linux Scripts

A collection of Linux shell scripts developed to automate common system administration tasks.

## 📌 Features
- File execution automation
- Directory listing and management
- File modification utilities

## 📂 Scripts Included

### 1. execute.sh
- Automates execution of commands/scripts

### 2. listingfile.sh
- Lists files and directories with detailed information

### 3. listmodify.sh
- Performs file modifications and operations

## ⚙️ Technologies Used
- Bash Shell Scripting
- Linux Commands (ls, chmod, file handling)

chmod +x script_name.sh
